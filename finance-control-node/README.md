## Descrição
Serviço de gerenciamento de investimentos.

## Instalação

```bash
# Instalando dependencias do projeto
$ npm install

# Conectando prisma ao projeto
$ npx prisma generate
```

## Rodando o APP

```bash
# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

- Conta
  - Criar contas
  - Editar contas
  - Deletar contas
  - Listar contas
  - Buscar conta por id

- Ativo
  - Criar ativos
  - Listar ativos
  - Editar ativos
  - Deletar ativos
  - Buscar ativo por id

- Operações
  - Realizar operações
  - Editar operações
  - Deletar operações
  - Listar operações
- Dashboard
  - Graficos
    - Ativo
    - Setor
    - Subsetor
    - Corretora*
  - Evolução patrimonial*
